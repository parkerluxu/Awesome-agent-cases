# AI Agent 办公自动化案例

> 把日常办公与个人效率交给智能体：邮件、文档、日程、会议纪要，以及笔记、知识管理与学习工作流。

本页是《Awesome-agent-cases》的办公自动化分册，覆盖「日常办公生活」场景里可落地的教程、开源项目、产品与工具。每日时效新闻见 AI 日报（本仓库 `daily_news/`）。

## 收录范围

- **办公工作流自动化**：邮件、文档、表格、日程、会议纪要等跨应用重复劳动
- **RPA + Agent**：跨应用自动化与流程编排（n8n、Zapier、Make 等）
- **个人效率与知识管理**：第二大脑、笔记自动化、学习 Agent、信息整理
- **生产力套件 AI**：Copilot、Google Workspace AI、Notion AI 等内嵌式智能体

## 如何使用

下文「资料更新」区块由脚本根据搜索、RSS、GitHub 与产品/媒体反馈自动追加，按内容类型（项目 / 论文 / 教程 / 产品）分组，作为本主题的新发现资料线索。

> 说明：本页专注「长效精选资源」，与 AI 日报中零散提到的办公 Agent 新闻互为补充，不重复。
